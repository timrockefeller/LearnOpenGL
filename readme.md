### 碎碎念

深知自身Cpp的薄弱，故学习了Ubpa的开发架构，籍以加深理解。

本项目用于：

- 学习C++部分新特性的使用
- 学习OpenGL至渲染的底层过程
- 了解GLSL的编写
- 学习CMake创建工程的方式

在repo中可以不经意间看到：

- 似疯犬似野马的注释
- 绞尽脑汁抽象却没成功的类
- 由于库特性导致的面向过程瞎子

### 部署

- CMAKE
- VS Code & Cmake plugins
- gdb

相关安装详见[此处](https://blog.iik.moe/2019/08/19/unsorted/cmake-with-vscode/)。
